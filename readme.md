# Mentionlinks

Make the official links on Discord look unique.

List of currently supported links:

-   Discord
-   Spotify
-   Reddit
-   Twitter
-   Facebook
-   Instagram
-   Behance
-   Github
-   YouTube
-   Twitch
-   Netflix
-   DeviantArt
-   Dribble
-   Imgur

This is a fork from CreArts that hasn't been abandoned.
